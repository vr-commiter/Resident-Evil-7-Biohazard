# Resident Evil Biohazard Mod

## Installation

There are two versions of this mod available:

### Option 1: TrueGear Official Version

**base on:** [reframework](https://github.com/praydog/REFramework)  

Click **OK** to start the installation
![](https://static.truegear.tech/GitImg/General/ok.png)

### Option 2: Nexus Mods Version

**Created by:** charlotteliuok

1. Go to the mod download page: [Nexus Mods - Resident Evil 7 #282](https://www.nexusmods.com/residentevil7/mods/282)
2. Follow the instructions on that page to complete the installation
3. Once installed, click the **✕** in the top-right corner to close this dialog
![](https://static.truegear.tech/GitImg/General/close.png)

> **Note:** The Nexus version does not require any setup inside TrueGear_Player. After a successful installation, TrueGear_Player will **not** show it as installed — this is expected behavior.
